  • Paragraph tag
    • Element/Concept Name: <p> </p> (Paragraph)

    Purpose: Used to define a paragraph of text.

    Syntax: <p>This is a paragraph.</p>

    Example in context: 

    
                <body>
                <h1>My First Page</h1>
                <p>This is an example paragraph below the heading.</p>
                </body>
    Notes:

    - Browsers automatically add spacing before and after a <p>.

    - Do NOT put other block elements (like <div> or <h1>) inside a <p>.

    - If you need to add a line break within a paragraph, use the <br> tag.
